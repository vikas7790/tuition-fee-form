CEA (Tuition Fee) Reimbursement Form
=====================================

A self-contained, mobile-friendly HTML form for the Children
Education Allowance reimbursement proforma.

ONLINE USE
----------
After deployment, open the Vercel website link in Safari/Chrome/Edge. This is
the easiest mobile option because auto-save works under the website address.

iPhone tip: open the website in Safari, tap Share, then tap Add to Home Screen.
After that it opens like a normal app from your Home Screen.

HOW TO USE
----------
1. Double-click  index.html  to open it in your browser (Chrome/Edge recommended).
2. Fill in the fields. Dropdowns have an "Other…" option where free text is needed.
3. Employee name / Emp ID / Designation entered on Page 1 auto-fill the signature
   blocks on Pages 2 and 4, and the first child's details prefill Pages 3 and 4.
4. The total amount claimed is computed automatically from the expenditure rows.
5. Your entries are auto-saved to this browser/device (localStorage). If you
   are opening through WhatsApp, also use  "Save Copy"  because WhatsApp may
   open the file in a temporary viewer where auto-save is not reliable.
6. Click  "Print / Save PDF"  at the bottom. In the print dialog choose
   "Save as PDF" as the destination, or pick a physical printer. Output is
   formatted as 4 A4 pages matching the original government proforma.
7. If a mobile browser does not open the print dialog, tap  "PDF View" .
   It opens a clean print-only copy. From that screen use the browser menu
   or Share button, then choose Print / Save as PDF.
8. Tap  "Save Copy"  after filling details to download/save a new HTML file
   that has your entered data inside it.
9. Click  "Clear"  to reset everything.

MOBILE
------
Open the file on your phone in Chrome, Edge, or Safari. Layout reflows to a
single column with tap-friendly inputs and native date pickers. For PDF output,
first try  Print / Save PDF . If nothing opens, tap  PDF View  and use the
browser's Share / Print / Save as PDF option from that print-only page.

Important: WhatsApp often opens HTML attachments inside a temporary preview.
In that mode auto-save may not survive reopening. Best option is to save the
HTML file to your phone first, open it in Chrome/Edge/Safari, and tap
Save Copy after filling the form.

NOTES
-----
- No internet connection is needed — everything runs locally.
- No data is sent to any server. All values stay on this device.
- The hosted website can also work offline after it has opened once.
- This folder is independent of any other project.
